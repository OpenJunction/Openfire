Jingle Nodes Plugin Readme

Overview

Implementation of http://xmpp.org/extensions/xep-0278.html. Provides a NIO Implementation of Jingle Nodes, which is an easy to NAT Traversal service purely based on XMPP and aims mainly, but not limited to, Jingle Clients.

Installation

Make sure Openfire is running on a Public IP interface and with Firewall opened for UDP Traffic both ways on port range 30000 to 50000.
